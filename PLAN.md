- 해당 지역에 작품 기록된 경우 시/군/구 중 '구'단위로 하나씩만 동그라미 표식 띄우기(동일 시에 작품 여러개여도 하나만, 만약 단위 면 등 다르면 해당 단위에 맞게). 그리고 해당 동그라미 누르면 해당 지역에 메모된 작품들 중 랜덤으로 하나 뜨도록. 그리고 시보다 더 큰 단위로 작품 위치 메모된 경우 어떻게 해야 가장 UIUX에 좋을지 논의
- UI UX 최적화. 디자인 변혁, 버튼, 컴포넌트 디자인, 위치 등 최적화
- 인기도 도입. 인기도에 따라 동그라미 표식 색상 차이 두도록. 이때 인기도는 사용자들이 평가(SNS 좋아요 처럼 하트로 표시)
- 지도에서 대분류(시) 클릭 시에도, 해당 시에 등록된 작품들 뜨도록 하기
- 이야기들 적고 공유하고, 추천, 사진 업로드 등 게시글 관련 디벨롭
- 배포
- 사용자들이 실제로 많이 사용할 수 있게, 경쟁력 있게 하려면. 현실적 유입
- 앱에서 실제 방문으로 이어지는 동선을 어떻게 제공할지
- 목적 없이 들어온 사용자가 빠져들 수 있는 콘텐츠 소비 흐름
- 사용자가 실제로 접근할 수 있는 URL과 설치 가능한 PWA
